# DesafioConnectSeaFrontEnd

Este projeto foi gerado utilizando Angular CLI versão 20.3.10.

## Como executar o front-end

Para iniciar o servidor de desenvolvimento, utilize o comando:

```
ng serve
```

Depois que o servidor estiver em execução, acesse no navegador:

http://localhost:4200/

A aplicação recarrega automaticamente sempre que os arquivos forem modificados.

## Gerar componentes

Para criar um novo componente, execute:

```
ng generate component nome-do-componente
```

Para ver todos os schematics disponíveis:

```
ng generate --help
```

## Build do projeto

Para gerar o build otimizado na pasta dist:

```
ng build
```
